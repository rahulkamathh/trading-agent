Setup — send to a group chat or personal DM:
1. Create a bot: Message @BotFather on Telegram → /newbot → copy the token
2. Add bot to your group: Invite the bot into the Telegram group you want alerts in (or DM it directly)
3. Get the chat ID: Send any message in the group, then open:
   https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
   Look for "chat":{"id":-100XXXXXXXXX} — group IDs are negative numbers
4. Add to Railway Variables:
   TELEGRAM_BOT_TOKEN = your bot token
   TELEGRAM_NOTIFY_CHAT_ID = the group/DM chat ID
Optional SMS: add TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM_NUMBER, TWILIO_TO_NUMBER

How dynamic risk management works:
Position sizing: Each trade risks exactly 0.5% of portfolio value. Size = risk ÷ ATR × conviction. No hardcoded amounts.
Event calendar: MSCI rebalancing (quarterly), F&O expiry (monthly), RBI MPC days, Budget → auto-reduces position sizes 2–3 days before.
India VIX: VIX > 20 → HIGH risk, > 30 → EXTREME. Positions shrink proportionally.
Drawdown circuit-breaker: Portfolio down 2% from today's peak → CAUTION (half size). Down 4% → STOP (no new buys). Down 6% → HALT.
EXTREME macro: Zero new buys until risk normalises. Existing positions held until SL/TP.
